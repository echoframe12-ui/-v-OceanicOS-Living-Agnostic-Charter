(`oceanic_os.py`) and `/observer` both speak this, from the single source in
[`identity.py`](identity.py) — the system says the same name of itself
everywhere.

```
/
└── Ω∞v Compiler

        └── Living Agnostic Charter
```

| Ω∞v Compiler | the verification compiler — attest, don't assert |
