## Verification Protocol

All new capabilities must undergo the following verification before being added to the registry:

1.  **Charter Alignment**: Does it respect the 10 Constitutional Principles?
2.  **Architectural Fit**: Does it belong to the designated Layer/Stage?
3.  **Documentation**: Is the logic and usage clearly recorded?
4.  **Provenance**: Is the history of its creation preserved?
